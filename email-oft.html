HTML File Creation

  1. Copy the respective email and paste _HTML on the end of the name.
  2. Change the Email Group to Events and turn off Enable Eloqua Email Tracking.
  3. In Email Style change Background Color to white.
  4. Replace header to AP – Blank Header with Background Color to white.
  5. Uncheck all the Manage Links.
  6. Remove the entire line if it contains any Field Merge.
  7. Generate html field with its respective UTM Code Link .
  8. Remove the auto generated date added before the email name.

HTML Edits

  1. Remove the empty table which used to contain Header.
  2. Replace width=”100%” to width=”600”.
  3. Add align="center" on the first table tag after the body tag.
  4. Replace: <span><eloqua type="emailFooter"></eloqua></span>   with it's respective email footer source code and change Background colour to bgcolor="#ffffff".
  5. Remove the entire line if it contains any Field Merge and Unsubscribe Link in Footer.
  6. Remove &elqTrack=false and Add the UTM Contents on end of the links.

    • https://www.example.com?&utm_medium=email&utm_content=AP_*_HTML&utm_source=AP_*_HTML&utm_campaign=UTMCode
    • https://www.example.com?&utm_medium=email&utm_content=AP_*_OFT&utm_source=AP_*_OFT&utm_campaign=UTMCode
  7. No need to track Email just Remove &elqTrack=false.

Changing HTML to OFT

  1. In Eloqua => Upload an Email > Upload the edited HTML file.
  2. Next => Check image
  3. Next => Make shore all links are Unchecked
  4. Next =>
    • Name            : Change _HTML.html to _OFT
    • Folder           : Direct to the folder path.
    • Email Group : Events (most of the time)
