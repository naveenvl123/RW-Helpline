HTML File Creation

  1. In Eloqua Copy the respective email and paste _HTML on the end of the name.
  2. Change the Email Group to Events and turn off Enable Eloqua Email Tracking.
  3. In Email Style change Background Color to white.
  4. Replace header to AP - Blank Header with Background Color to white.
  5. Uncheck all the Manage Links.
  6. Remove the entire line if it contains any Field Merge.
  7. Generate html file with it's respective UTM Code Link .
  8. Remove the auto generated date added before the email name.

  4. Next =>

    • Folder           : Direct to the folder path.

  5. Settings:

    • Email Header : AP - Blank Header
    • Email Footer  : AP - Blank Footer

    • Uncheck        : Enable email tracking
