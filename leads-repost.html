Leads Repost Creation

  1. In Eloqua forms Clone AP_*_ContactUs_LeadsRepost_2024_04.
  2. Make shore the Form Name and HTML Name are the same.

    • Form Name  : AP_WB_Smart_Webinar_ContactUs_LeadsRepost_2024_04
    • HTML Name : AP_WB_Smart_Webinar_ContactUs_LeadsRepost_2024_04
      • Form HTML name must be below 50 characters.
  3. For CRM GUID:

    • Paste the value on Campaign GUID (AP) > Static Value.
  4. For Lead Name and Lead Description:

    • Click Processing button near Save.
    • On the 3rd Data Object Paste the values there.

      • Topic(Lead Name)
      • Lead Description
    • The Lead Name and Description must be pasted as it is without making any changes.
